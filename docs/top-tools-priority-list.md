# Top 25 Tools — Promptly Score v2.2 Audit Priority

**Purpose.** The first 25 tools to re-review under the **Promptly Score v2.2 standard**
(`~/Documents/methodology`). This is a *prioritised queue only* — **no v2.2 scoring has been
performed**. The "Current Promptly Score" column shows each tool's **live** figure today
(the v2.1-style weighted average of synthetic placeholder pillars) purely as a baseline to
measure movement against; it is **not** a v2.2 score.

**How the 25 were chosen.** Priority weighting follows the v2.2 docs:
1. **Visibility / reputational exposure** — homepage- and category-featured tools are seen most, so an inaccurate score there costs the most trust.
2. **Likelihood of movement under v2.2** — the new **graduated cap** (`composite ≤ min(Safeguarding, Privacy) + 1.0`), the **Avoid trigger** (Safeguarding *or* Privacy < 4.5), and the **8.5 desk-review ceiling** on Safeguarding/Privacy will pull some current scores down. General LLMs and student-facing tools are most exposed.
3. **Safeguarding / Data-Privacy criticality** — student-facing AI, wellbeing, and safeguarding/MIS systems carry the highest legal-barrier risk.
4. **Migration queue** — the four "ready rows" already staged in the migration workbook (Diffit, Curipod, Century Tech, Satchel One). MagicSchool AI already carries real sub-scores (Brand Bible v1.1) and is the calibration anchor.

**Visibility key:** *Homepage* = referenced on the homepage role widget · *Category Featured* = a category leader surfaced in its directory · *Directory Only* = listed but not featured.
