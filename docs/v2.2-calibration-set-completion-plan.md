cap-binding case, ≥1 Avoid (privacy), ≥1 Avoid (safeguarding), ≥1 N/A-rescale case, and ≥1
desk/hands-on contrast — each scored with named reviewer, cited evidence, basis and (where
triggered) a second reader.
